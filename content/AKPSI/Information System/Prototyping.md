# Prototyping
#akpsi #note #information-system-introduction 
Class: [[Information System Analysis and Design Class (AKPSI)]]
Topic: [[Information System]]

## Definition
Creating a working model that can evolve into the final system.

## When to use
  - **Iterative Development**: When the system requirements are unclear and may evolve over time.
  - **User Feedback**: When frequent user feedback is needed to refine the system.
  - **Complex User Interfaces**: When the system has complex or innovative user interfaces that need to be tested and refined.
