# FireWire (IEEE 1394, Apple)

- **Low-cost digital interface**
- **Transfer speeds**: Up to **800 Mbps**
- **Hot pluggable**
- Supports two types of data connections:
  - **Asynchronous connection**
  - **Isochronous connection**
